You-chromeplete-amazon

Chrome extension for easy Amazon product review search on YouTube

Download this project as a .zip file Download this project as a tar.gz file

What is it?

You-Complete-Amazon is a Chrome extension that adds a button to Amazon product pages. Clicking on the button searches YouTube for product reviews.
